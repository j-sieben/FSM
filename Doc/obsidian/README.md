# FSM-Dokumentation

Dieses Verzeichnis ist ein Obsidian-Vault für die konzeptionelle Dokumentation des FSM-Frameworks.

Zum Öffnen in Obsidian dieses Verzeichnis als Vault auswählen. Der Einstieg erfolgt über [[00_Start/Startseite|Startseite]]. Die generierte Natural-Docs-Referenz liegt eine Ebene höher unter `Doc/index.html` und ergänzt diesen Vault um Details zu einzelnen Datenbankobjekten.

Die Markdown-Dateien sind eigenständig lesbar. Obsidian löst zusätzlich Wiki-Links und Mermaid-Diagramme direkt auf.

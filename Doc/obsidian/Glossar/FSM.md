# FSM

Eine Finite State Machine beziehungsweise ein endlicher Zustandsautomat beschreibt den Lebenszyklus eines Objekts als Menge von [[Glossar/Status|Status]] und zulässigen [[Glossar/Transition|Transitionen]]. Im Framework bezeichnet `FSM` außerdem das zentrale PL/SQL-Package, das diese Übergänge ausführt.

## Im Projekt

Das Framework hält den zulässigen Graphen in Metadaten und die fachliche Entscheidung in konkreten PL/SQL-Handlern.
